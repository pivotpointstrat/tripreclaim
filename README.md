# TripReclaim

> Post-booking flight price drop recovery service

TripReclaim monitors your flight price after booking. When it drops, we alert you instantly so you can claim a refund.

**Live site:** https://vs-a4-reclaim-0ejs.netlify.app

## Stack
- Frontend: Static HTML/CSS (deployed on Netlify)
- Payments: Stripe
- Email: Resend
